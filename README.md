# Home Automation

A modern, beautiful web interface for home automation, integrating multiple smart home platforms into a unified dashboard.

## Integrations

- **Fibaro HC3** - Z-Wave home controller (lights, blinds, scenes, sensors)
- **Netatmo** - Weather station and indoor climate monitoring
- More integrations planned

## Tech Stack

- **Runtime**: Node.js with TypeScript
- **Frontend**: React with Vite
- **Backend**: Express.js REST API
- **Styling**: Tailwind CSS

## Getting Started

```bash
npm install
npm run dev
```
